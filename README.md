# Bramble & Branch Counseling

Website for Bramble & Branch Counseling — Kerith Hopper, LCSW.

## Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Playfair Display + Inter
- **Hosting**: Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Pushes to `main` auto-deploy via Vercel.
